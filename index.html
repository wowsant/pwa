Um Progressive Web App é:

Características de uma aplicação web progressiva

Progressivo

Para qualquer usuário, independentemente do browser.

Responsivo

Feito para qualquer dispositivo: desktop, tablet e mobile.

Conexão

Funciona mesmo se o usuário estiver offline.

App-like:

não é necessário baixar atualizações do aplicativo, o browser simplesmente irá detectar e atualizar automaticamente, caso necessário;

Seguro

Somente com https.

Engajável

Através de push notifications, o usuário pode ser constantemente engajado.

Instalável

é possível adicionar um ícone na tela principal do smartphone com apenas um clique.
